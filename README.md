# File Uploader Backend

A minimalist backend service built with Express.js to practice file handling and database management using the Prisma ORM.

---

## Features
* Secure file uploads via Multer middleware.
* Metadata storage in a PostgreSQL database via Prisma.
* Basic CRUD operations for uploaded file records.

## Tech Stack
* Runtime: Node.js
* Framework: Express
* PostgreSQL Database Toolkit: Prisma ORM
* Storage: Local file system

---
